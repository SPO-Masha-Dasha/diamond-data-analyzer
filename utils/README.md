Приложение для анализа данных о бриллиантах с графическим интерфейсом на PyQt5.





**Основные возможности:**



1\. Статистика данных

&nbsp;  - Основная статистика (среднее, медиана, стандартное отклонение)

&nbsp;  - Информация о типах данных и размере набора

&nbsp;  - Анализ пропущенных значений

&nbsp;  - Уникальные значения по колонкам



2\. Графики корреляции

&nbsp;  - Визуализация взаимосвязей между числовыми параметрами

&nbsp;  - Графики корреляций с использованием seaborn

&nbsp;  - Возможность выбора конкретной переменной для анализа



3\. Тепловая карта

&nbsp;  - Корреляционная матрица с цветовой кодировкой

&nbsp;  - Настройка цветовых схем (coolwarm, viridis, plasma и др.)

&nbsp;  - Отображение числовых значений корреляций



4\. Линейные графики

&nbsp;  - Построение графиков по любым числовым столбцам

&nbsp;  - Различные типы графиков: точечные, линейные, гистограммы

&nbsp;  - Добавление линий тренда и расчет корреляции



5\. Лог действий

&nbsp;  - Автоматическое логирование всех операций

&nbsp;  - Временные метки для каждого действия

&nbsp;  - Возможность очистки лога



6\. Загрузка данных

&nbsp;  - Автоматическая загрузка diamonds.csv при запуске

&nbsp;  - Поддержка форматов CSV и Excel

&nbsp;  - Диалоговое окно выбора файла

Для того, чтобы скачать данные нужно нажать на кнопку "файл" в верхнем левом углу экрана и выбрать кнопку "загрузить данные"





**Первый запуск:**



Приложение автоматически загрузит файл diamonds.csv

Данные появятся во всех вкладках

Статусная строка покажет количество загруженных записей





**Структура проекта:**
DiamondApp/

├── main.py                 # Главный файл приложения

├── diamonds.csv            # Пример данных о бриллиантах

├── README.md              # Документация

├── requirements.txt       # Зависимости проекта

├── widgets/              # Модули вкладок

│   ├── stat\_tab.py       # Вкладка статистики

│   ├── correlation\_tab.py # Графики корреляции

│   ├── heatmap\_tab.py    # Тепловая карта

│   ├── linear\_tab.py     # Линейные графики

│   └── log\_tab.py        # Лог действий

└── utils/                # Вспомогательные модули

&nbsp;   └── \_\_init\_\_.py       # Инициализация пакета





**Запуск приложения:**


1. Клонируйте репозиторий или скачайте файлы проекта

2\. Перейдите в папку проекта:



bash

cd DiamondApp



3\. Запустите приложение:



bash

python main.py
