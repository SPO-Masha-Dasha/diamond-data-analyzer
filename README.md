Приложение для анализа данных о бриллиантах с графическим интерфейсом на PyQt5.


Основные возможности:

1. Статистика данных
   - Основная статистика (среднее, медиана, стандартное отклонение)
   - Информация о типах данных и размере набора
   - Анализ пропущенных значений
   - Уникальные значения по колонкам

2. Графики корреляции
   - Визуализация взаимосвязей между числовыми параметрами
   - Графики корреляций с использованием seaborn
   - Возможность выбора конкретной переменной для анализа

3. Тепловая карта
   - Корреляционная матрица с цветовой кодировкой
   - Настройка цветовых схем (coolwarm, viridis, plasma и др.)
   - Отображение числовых значений корреляций

4. Линейные графики
   - Построение графиков по любым числовым столбцам
   - Различные типы графиков: точечные, линейные, гистограммы
   - Добавление линий тренда и расчет корреляции

5. Лог действий
   - Автоматическое логирование всех операций
   - Временные метки для каждого действия
   - Возможность очистки лога

6. Загрузка данных
   - Автоматическая загрузка diamonds.csv при запуске
   - Поддержка форматов CSV и Excel
   - Диалоговое окно выбора файла
Для того, чтобы скачать данные нужно нажать на кнопку "файл" в верхнем левом углу экрана и выбрать кнопку "загрузить данные"


Первый запуск:

Приложение автоматически загрузит файл diamonds.csv
Данные появятся во всех вкладках
Статусная строка покажет количество загруженных записей


Структура проекта:
DiamondApp/
├── main.py                 # Главный файл приложения
├── diamonds.csv            # Пример данных о бриллиантах
├── README.md              # Документация
├── requirements.txt       # Зависимости проекта
├── widgets/              # Модули вкладок
│   ├── stat_tab.py       # Вкладка статистики
│   ├── correlation_tab.py # Графики корреляции
│   ├── heatmap_tab.py    # Тепловая карта
│   ├── linear_tab.py     # Линейные графики
│   └── log_tab.py        # Лог действий
└── utils/                # Вспомогательные модули
    └── __init__.py       # Инициализация пакета


Запуск приложения:

1. Клонируйте репозиторий или скачайте файлы проекта
2. Перейдите в папку проекта:

bash
cd DiamondApp

3. Запустите приложение:

bash
python main.py
